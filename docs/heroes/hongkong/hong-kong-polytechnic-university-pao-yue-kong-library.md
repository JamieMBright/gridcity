## Description
university library

Pao Yue-kong Library, full name Pao Yue-kong Library, the Hong Kong Polytechnic University, or PYK Library in short, is the main library of Hong Kong Polytechnic University (PolyU). Established as the Hong Kong Polytechnic Library in 1972, it is now a modern academic library with 3,600 seats and a collection of 1,100,513 hardcopy books, 8,117,470 e-Books and, 255,412 e-Journals (titles). The Library Extension Project won the 2022 Best of Design Award by the Architect’s Newspaper.
