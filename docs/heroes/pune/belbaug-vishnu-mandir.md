# Belbaug Vishnu Temple
*temple · 1761*

Reference: real Pune landmark (research compiled for the bespoke-hero sprite).
Wikidata/Wikipedia: search "Belbaug Vishnu Temple Pune".

## Description
A Vishnu (Vyankatesh/Balaji) temple built in the Belbaug enclosure by the Peshwas in the 18th century, with a stone mandap, a carved wooden ceiling and a tiered shikhara; once set in a garden of bel trees from which it takes its name.
