# Lakdi Pul (Sambhaji Bridge)
*bridge · 1885*

Reference: real Pune landmark (research compiled for the bespoke-hero sprite).
Wikidata/Wikipedia: search "Lakdi Pul (Sambhaji Bridge) Pune".

## Description
Literally 'the wooden bridge', a historic crossing of the Mutha at Deccan, long since rebuilt in stone and concrete (officially Sambhaji Bridge) but still universally called Lakdi Pul; a busy multi-arch road bridge with parapet lamps.
