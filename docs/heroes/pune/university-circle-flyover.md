# University Circle Flyover
*flyover, infrastructure · 2018*

Reference: real Pune landmark (research compiled for the bespoke-hero sprite).
Wikidata/Wikipedia: search "University Circle Flyover Pune".

## Description
The multi-level grade-separated interchange at the University Circle (Ganeshkhind) junction, a stacked concrete flyover on tall piers carrying the Aundh-Shivajinagar and University-road traffic; a major node of the city's road network.
